# Object Change: GroundHeatExchanger:System

A new method for sizing ground heat exchangers was added in 25.2.
To accompany this change, a new optional field was inserted into the GroundHeatExchanger:System object.
The first 10 fields stay the same.
A new 11th field is inserted called "GHE:Vertical:Sizing Object Name".
This can be left blank when upgrading files from 25.1 to keep the same behavior as before.
